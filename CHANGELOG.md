# Changelog

## 1.0.0

- Initial production-focused release of Fatsan Ore Generation
- Folia-first ore-only generation architecture using `BlockPopulator` + `LimitedRegion`
- Paper compatibility for `1.21.x`
- Immutable hierarchical rule snapshot system
- Per-profile, per-environment, per-world, and per-biome rule overrides
- Bilingual config comments and language files (`en_US`, `tr_TR`)
- Admin commands for reload, info, inspect, stats, and rule dumping
- Unit coverage for rule precedence and ore generation/removal behavior
